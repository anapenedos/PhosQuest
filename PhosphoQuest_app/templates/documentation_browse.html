Browse and Search functionality

PhosphoQuest users can use the search function to search for Kinases, substrates or inhibitors by accession number or name. The search options include like or exact match searches.

'Like' match searches are not case sensitive and will find results with text matches inside other words, for example "cellular" in "Extracellular-matrix".

'Exact' match searches are case sensitive and will only find results with text that exactly matches that in the database field.
